별표가 중요한거지 args랑 kwargs는 관례구나

*랑 ** 이 별표가 중요한거지 뒤에 글자는 뭘 써도 괜찮단다. 이제 함수 안에 인자를 몇 개를 널지 확실하지 모를 때 쓰는 착한 친구들이다. 생긴게 무섭게 생겨서 그렇지, 일단 보자. 

class MyFunc:
    def __call__(self, *varA, **varB):
        print("호출됨")


f = MyFunc()
    
f(55)    
    
[output]  

호출됨
    
f(ebebebe = "1515")   
    
[output] 

호출됨
    
f(a.pick(), 5555, "asdsad", ebebebe = "1515")
    
[output] 

호출됨

인자가 몇개가 오든 누가 오든 대모님처럼 다 받아주신다.

근데 *arg는 키워드 되지 않은 가변 갯수의 인자를, **kwargs는 키워드 된 가변 갯수의 인자를 받는단다. 무슨 말인지 코드로 보자. 

class MyFunc:
    def __call__(self, *var):
        print("호출됨")


f = MyFunc()    
    
f('dd')      
    
[output] 

호출됨    
    
f(name = 'dd')    
    
[output] 

TypeError: __call__() got an unexpected keyword argument 'name'
    
class MyFunc:
    def __call__(self, **kwargs):
        print("호출됨")


f = MyFunc()    
    
f('dd')    
    
[output] 

TypeError: __call__() takes 1 positional argument but 2 were given
    
f(name = 'dd')    
    
[output] 

호출됨
